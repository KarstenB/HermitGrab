What is HermitGrab?

HermitGrab is a lightweight dotfile manager built in Rust. It’s designed for ease of use and rapid setup, using a powerful tag-based system to manage configuration files for different environments and use-cases.

Key Features

  • Configuration with hermit.toml
  • Installation of tools
  • Batteries included, contains: Universal binary installer (ubi)
  • Automatic discovery of config repositories via GitHub/GitLab topics
  • GitHub Device Login for easy remote access
  • Single static binary, easy to install and distribute
  • Supports symlinking, hardlinking, copying, and patching

Example Configuration

Here's an example hermit.toml for installing and configuring the fish shell:

# This configuration file provides the fish shell
provides = ["fish"]
requires = ["+os_family=unix", "-zsh"]

[[file]]
source = "config.fish"
target = "~/.config/fish/config.fish"
fallback = "BackupOverwrite"

[[file]]
source = "functions/egrep.fish"
target = "~/.config/fish/functions/egrep.fish"
requires = ["+ripgrep"]

[[patch]]
type = "JsonMerge"
source = "vscode_settings.json"
target = "~/.vscode-server/data/Machine/settings.json"
requires = ["user=vscode"]

[[install]]
name = "fish"
check_cmd = "command -v fish"
post_install_cmd = "fish --install < {{ hermit_this_dir }}/confirm.txt"
source = "ubi"
requires = ["+arch=aarch64", "+os=linux"]

[install.variables]
exe = "fish"
url = "https://github.com/fish-shell/fish-shell/releases/download/4.0.2/fish-static-aarch64-4.0.2.tar.xz"

Installation

Make the binary executable and place it in your PATH:

chmod +x hermitgrab
mv hermitgrab ~/.local/bin

You can now run hermitgrab from anywhere.
